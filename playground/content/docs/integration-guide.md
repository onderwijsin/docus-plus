This application deliberately keeps content and branding small while enabling the Docus Plus integration surface.

## Covered features

- The Docus Plus Nuxt layer and its shared component contract.
- Multi-source Scalar API references at `/api-reference/yaml` and `/api-reference/json`.
- Nuxt Content, MCP, assistant, SEO, analytics, Turnstile, healthcheck, and cache module configuration.

Use the official documentation or `.starters/default` when you need a fuller implementation example.
